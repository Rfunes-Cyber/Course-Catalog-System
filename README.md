# Course-Catalog-System
- Course catalog system written in C using structs and binary file I/O

# Description
- This project simulates a course catalog management system using binary file storage and CRUD operations.

## Features
- Create course records
- Read existing course records
- Update course information
- Delete course records
- Persistent binary file storage

## Technologies Used
- C Programming
- GCC Compiler
- Structs
- Binary File I/O

## How to Compile
```bash
gcc assign4.c -o assign4
```

## How to Run
```bash
.\assign4.exe
```
